# mac-changer-tool
In this python project, you'll be able to change you mac address to another onem but it usable in linux OS only because the system commands inside the program are using in native system commands, but remember, this going to change you mac address in memory not physical, that's mean once you restart your machine, it's going to retreive to original mac unique mac address.
 To ise the tool, just execute this command:
 >python3 main.py --interface [interface name] --mac [new mac address]
 then, press enter to run the command.
